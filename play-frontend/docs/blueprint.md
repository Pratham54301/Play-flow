# **App Name**: PayFlow

## Core Features:

- Name Input: Name field for collecting user's name.
- Email Input: Email field with validation to ensure correct format.
- Mobile Number Input: Mobile number input field for collecting user's phone number.
- Amount Input: Input field for the payment amount.
- Pay Now Button: "Pay Now" button to trigger payment processing.
- Labels and Placeholders: Clear labels and placeholders for all input fields.
- Validation message: Show error message when there is an error

## Style Guidelines:

- Primary color: Muted indigo (#5C6BC0) to give a sense of calm reliability.
- Background color: Light gray (#F0F2F5), providing a clean, unobtrusive backdrop.
- Accent color: Soft lavender (#C5CAE9) for subtle highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif for clear, modern text. 
- Responsive layout adapting to different screen sizes.
- Use subtle transitions and animations on button hover states.
